## 2. 개발 환경 및 기술 

> **Client**
> * Kotlin - Android Stuio
> * MVVM 패턴 
> * Paging3 
> * Coroutine
> * Retrofit2 
> * CustomView
> * LiveData
> * DataBinding
> * ViewPager
> * Fragment
> * Room Database
> * FCM / Googel Drive Api / Kakao Map Api 

> **Server**
> * Java8 - SpringBoot - InteliJ  
> * AWS EC2, ROUTE53
> * Jsoup 
> * FCM 
> * Lombok

> **DataBase**
> * MySQL

</br>

## 4. 핵심 기능 

> JNUCenter의 핵심 기능은 2가지입니다. 

</br>

> * 부서명으로 건물 위치 / 걸리는 시간(걸어서, 뛰어서) / 장소 힌트 맵 표시  
> * 비대면 강의 긴급 기한 알람

### 4.1.1 개발 의도 
  </br>
  학교 건물이 너무 많아서 시험장이나 특정 부서를 찾아갈 때의 혼란을 줄이기 위해 만들었습니다.</br> 
  특히 뛰었을 때, 걸었을 때 시간을 표시해 지각하지 않도록 편의를 제공했습니다.</br>
